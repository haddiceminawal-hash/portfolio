# Portfolio – Haddice Minawal

Mein persönliches Portfolio: Schule tagsüber, Code danach. Reines HTML, CSS &
JavaScript, ohne externe Bibliotheken.

## Features

- Terminal-Intro mit Typing-Animation
- Light/Dark-Toggle (merkt sich die Wahl, folgt sonst dem System)
- Scroll-Reveal-Animationen, `prefers-reduced-motion` wird respektiert
- Verlinkt meine anderen Projekte:
  [applecake](https://github.com/haddiceminawal-hash/applecake) ·
  [tictactoe](https://github.com/haddiceminawal-hash/tictactoe)

## Lokal ansehen

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

## Dateien

- `index.html` – Struktur (Hero, Projekte, Skills, Timeline, Kontakt)
- `style.css` – Design & Theming
- `script.js` – Terminal, Theme-Toggle, Scroll-Reveal
